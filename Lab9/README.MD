# Módulo 3 — Backend API
### EstateHub API · Node.js, Express & Prisma ORM

---

## Descripción

EstateHub API es una API REST para gestión de propiedades inmobiliarias construida con Node.js y Express. Permite realizar operaciones CRUD sobre propiedades, filtrarlas por distintos criterios y consultar estadísticas del inventario.

El backend corre en el puerto `3002` y se comunica con un frontend React en el puerto `3001`.

---

## Tecnologías

| Tecnología | Descripción |
|---|---|
| Node.js | Runtime de JavaScript |
| Express | Framework web para la API REST |
| TypeScript | Tipado estático |
| Prisma ORM | Acceso a base de datos (puente con SQLite) |
| SQLite | Base de datos local basada en archivo |
| Zod | Validación de datos de entrada |
